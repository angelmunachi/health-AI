# Health-AI: AI-Powered Leg Analysis

![Health-AI Logo](static/logo.png)  

**Health-AI** is an AI-powered web application that analyzes images of your leg to detect swelling and potential health issues. Built with **FastAPI**, **OpenAI**, and **Three.js** for interactive 3D visualization.
